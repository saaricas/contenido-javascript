FUNCIONES

PARTE 5: FUNCIONES

Las funciones son bloques de código, lo que quiere decir que son un conjunto de instrucciones cuyo objetivo es ejecutar una tarea concreta.

Las funciones pueden contar con varios inputs pero únicamente devolverán una respuesta (output)

Ventajas que tiene el uso de las funciones:

  • Evitar instrucciones duplicadas
  • Solucionar un problema complejo con tareas sencillas
  • Focalizar las tareas prioritarias para el programa
  • Aportar cierto orden y entendimiento al código
  • Aportar facilidad y rapidecez para hacer modificaciones

Function Declaration

Para declarar una función se utiliza la keyword function seguida del nombre que le queramos dar a la función. Entre paréntesis (), se establece los parámetros

Ejemplo de función:

function saludo(name){
return `hola ${name}`
}
saludo(Sara) -> Llamamos a la función para ejecutarla

Las funciones pueden incluir más de un parámetro. Para ello se coloca una coma , para separarlos

Otro aspecto importante de las funciones es que siempre hay que cerrarlas. Para ello se utiliza la keyword return

Se cierra con return porque es la forma que tenemos de decirle a JS que esa es la salida de la funció y así la almacene

Si lo hiciesemos con console.log no estaríamos almacenando el resultado, ya que esto únicamente nos muestra el resultado pero no lo guarda. Es decir, es como si nos mostrase el "fantasma" de lo que se ejecutaría con el programa si lo llegásemos a hacer.
