PARTE 1: INICIAR JS

Existen dos formas de iniciar javascript en nuestro proyecto:

  • A través del propio código. Para ello utilizamos las etiquetas "script"
  • La otra opción que tenemos es a través de un archivo .js. Habitualmente este archivo se denomina "main.js"

Para esta práctica utilizaremos la primera opción. El código quedará comentado en la propia web de tal forma que se pueda reutilizar en caso de ser necesario a través del botón inspeccionar en el navegador, o bien a través de la descarga del repositorio de GitHub.

PARTE 2: VARIABLES
Las variable de tipo primitivo.

Esto quiere decir que NO son objetos y que, por lo tanto, no pueden ser cambiados, lo que significa que son “passed by value”, es decir, que se le puede cambiar el valor. Son por tanto los siguientes:

  • String: es una cadena de caracteres. Se escriben entrecomillados " ". Se pueden encadenar varias string a través del operador + e imprimirlas con console.log

    Otra forma de encadenar strings es mediante el uso de los acentos hacia la izquierda ``. Esto se haría de la siguiente manera:

    Ejemplo:

    let frase1 = "hola";

    let frase2 = "¿Qué tal?":

    console.log(`${frase1} ${frase2});

    Resultado sería: hola ¿Qué tal?

  • Number: se escriben sin las comillas, ya que sino se convertirán en tipo string. Para poner números decimales se puede utilizar e-n para indicar la potencia. Ejemplo: 2e3 = 2000.
  • Boolean: poseen dos valores true o false. Los valores false equivalen a 0, " ", undefined, null o NaN.
  • Null: significa que no existe lo que estamos buscando.
  • Undefined: la variable se ha declarado pero no tienen ningún valor.
  • Symbol: próximamente
