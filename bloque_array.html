ARRAYS

PARTE 6: ARRAYS

Los array pertenecen a la clase "objeto".

Son un listado de datos de diferentes tipos dispuestos de forma ordenada

Pueden incluirse unos dentro de otros

Son más útiles que los objetos para crear órdenes u operaciones complejas.

Se declaran mediante los corchetes []

Los índices son los números de posicion que tienen los datos dentro del array. Se comienza a contar desde 0 y los datos entre sí se separan por comas ,

Acceder a los datos del array

Podemos acceder a un dato que hay dentro de un array llamándolo mediante su posición:

console.log(array[numeroPosicion])

También podemos acceder a un dato de un array que esta dentro de otro array.

let lenguajes = [["HTML", 5],["CSS", 3],["JS",4]] //queremos acceder al dato CSS console.log(lenguajes[1][0])
Reemplazar datos array

Podemos cambiar los datos que se encuentran dentro de un array realizando lo mismo que para reasignar una variable

let names = ["Sara", "Luna"] //Para hacer la modificación accedemos al dato que queremos reemplazar a través de su índice y escribimos el nuevo dato. names[1]="Freyja"
Diferentes formas de acceder y modificar los datos de los array
Hay 3 fromas diferentes para realizar cambios o leer los datos de los array. Estas son:
  • Mutator: devuelve un resultado que modifica el array
  • Iterator: Recorre los valores del array. Bucles
  • Accesor: devuleve una copia del arrey original con el cambio hecho
